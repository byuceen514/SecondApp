Software Design Document: Trail Mix

Created by Marlee Seat, Tyler Remund, and Robert Stevens

 

Technical Specifications

Software Details

The purpose of this project was to make a web app that would create an elevation profile for a drawn route. The script was written in PyCharm, uploaded to GitHub, and then published to a BYU server. The app was created by Robert Stevens, Marlee Seat, and Tyler Remund.

 

Compatibility and Limitations

The app has only been tested on Google Chrome. Other browsers may or may not be compatible. This app uses javascript, and requires a browser with javascript plugins/compatibility. A copy of the script may be obtained from Tyler Remund (tremund@gmail.com).

Software Design

Purpose

This app was developed to provide hikers with an easy to use tool to judge the difficulty of a certain route. The app allows the user to create their own trail anywhere within the state of Utah, receive a notification regarding the calculated difficulty of such route, and then view a profile of how the elevation would change along the route.

Structure

The script is divided into several HTML/ Javascript pages.

Base

All pages use the Base page. This page contains the Tethys page layout and custom sidebar..   

Home

This page contains the Javascript code to run the Trail Profiler application.

Help File/ Documentation

This section of code contains the HTML code to view the code the user is currently reading as well as the Help File code.

 

Files

The Script operates off of a basemap obtained from the Unites States Geological Survey (USGS) site. It also uses an elevation DEM of the state of Utah which had been previously uploaded to the server. The icon for the app was downloaded from the web and uploaded with the script to GitHub.

 

User Interface Elements

The primary user interface element is the interactive map that allows users to draw a polyline that will then be displayed as an elevation profile. This element allows users to draw a line anywhere in the state of Utah, and see the elevation profile of that line. In addition, the alert box that pops up notifies the user of the difficulty of the route.
